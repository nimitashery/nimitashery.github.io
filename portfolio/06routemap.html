    Route Map

    About: The assignment for this class project was to create a map for a cycling competition for anywhere in the world and display it on a shaded relief terrain map.
    While the bike route is imaginary, the terrain was created using a 10 million scale DEM of India. ArcGIS Pro was used for raster processing, Photoshop for land cover and textures, and Illustrator to add vector data and type.

    Tools: ArcGIS Pro, Illustrator, Photoshop
